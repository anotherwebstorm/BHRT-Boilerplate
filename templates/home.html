BRHT Boilerplate

Super Blast BRHT

A boilerplate for those who wanted to gather Backbone, RequireJS, Handlebars and Twitter Bootstrap. If you need to learn more about these libraries please refer to google

Download Project at Github

Backbone

Backbone gives structure to web applications by providing models with key-value binding and custom events, collections with a rich API of enumerable functions, views with declarative event handling, and connects it all to your existing API over a RESTful JSON interface.

RequireJS

RequireJS is a JavaScript file and module loader. It is optimized for in-browser use, but it can be used in other JavaScript environments, like Rhino and Node. Using a modular script loader like RequireJS will improve the speed and quality of your code.

Handlebars

Handlebars provides the power necessary to let you build semantic templates effectively with no frustration. Mustache templates are compatible with Handlebars, so you can take a Mustache template, import it into Handlebars, and start taking advantage of the extra Handlebars features.

Twitter Bootstrap

Sleek, intuitive, and powerful front-end framework for faster and easier web development.
